**Formula1_OS** is a feature-rich, immersive Terminal User Interface (TUI) that transforms your command line into a Formula 1 Pit Wall. Built with Python and `rich`, it replaces mundane system tasks with high-octane racing metaphors.

* **Boot Sequence:** Authentic FIA 5-light start procedure every time you launch.

## ğŸï¸ Race Engineer Tools
* **Track Maps:** ASCII layouts of major circuits (Monza, Spa, Silverstone, etc.).
* **Countdown:** Live timer to the next Grand Prix on the 2026 calendar.
* **Hall of Fame:** Complete database of every World Drivers' Champion since 1950.
* **Team Radio:** Randomly plays iconic radio messages from F1 history.
* **Paddock News:** Fetches the latest F1 headlines via RSS feed.

## ğŸ› ï¸ Core Utilities and Commands
* **Grid (`ls`):** File listing disguised as a starting grid.
* **Box (`cd`):** Navigation system with random tyre compound changes on directory switches.
* **Formation (`update`):** System update wrapper.
* **Race update (`next`):** Showing the specific no.of days till the next race. (race calendar updated till 2026)
* **Hall of Fame (`champions`):** Showing the list of all Formula 1 World Champions (till 2025)
* **ASCII Race Circuit (`map <name>`):** Showing the ascii image of all the racing circuits across the world. (Currently only 18 circuits)
* **Formula 1 News (`news`):** The latest news in Formula 1
* **Internet speedtest (`drs`):** Gives your internet speedtest, but with a Formula 1 twist
* **Meme radio messages (`quote`):** Some of the iconic radio messages in Formula 1 history.
